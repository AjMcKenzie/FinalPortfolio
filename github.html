Why Github is important

   Github is an online program that allows useres to download it to their desktops and then submit their work to the online verson. This allows them to download their code and work on it anywhere in the world (if they have internet to download the code). Using this also allows other users to work on the same code as you do at the same time. All they have to do is sync when they are done working on it then do a pull request for the newset code.
   At first it was confusing and hard to get used to. But now I like it and its nice to be able to work on my projects at school then continue them at home. It was nice to be able to work with other people on the same code and not have to worry about not having the newest code. I know that if I want to go into programming or web design I will need to know Github. All the people that do programming or web design or any other job like that needs to knows/already knows Github.
